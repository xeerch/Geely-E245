# 1.2.0

## Country

Australia

## Release Date

02/04/2026

## Official Source

[Available](https://a-ap.storyblok.com/f/3000753/x/e0d01fbb99/ota-update_ex5_jan-26.pdf)

## Release Notes

1. **Android Auto** will be available in both wired and wireless formats; air conditioning controls will conveniently remain visible to ensure you do not have to toggle between screens to adjust the temperature.<br>
2. A **Drive Setup** function will allow you to store your preferred driving settings:<br>
   • Drive Mode (ECO / COMFORT / SPORT)<br>
   • Energy Regeneration Level (Low / Medium / High / Auto)<br>
   • Lane Keeping Assist (ON / OFF)<br>
   • Emergency Lane Keeping Assistance (ON / OFF)<br>
   • Driver Fatigue Detection (ON / OFF)<br>
   • Speed Limit Alert (OFF / Blinking Icon / Blinking Icon & Sound Alert)<br>
   You can set the Steering Wheel Shortcut button to activate your saved settings. Only one profile can be set up as it is not linked to the User Profile or the Memory Settings.<br>

3. **Three Choices of the AVAS** (Acoustic Vehicle Alerting System) or Low Speed Pedestrian Warning will be available for you to choose which include the current tone (Classic) and the two tones available on the Starray EM-i (Milky Way & Constellation).<br>
4. **Tailgate Control** can now be added to the Swipe Down shortcut menu which includes Open, Close, and Pause functions. Note: this function is only available for Inspire Variants.<br>
5. Changing **Drive Modes** will now play a notification sound to confirm that the vehicle is in a new driving mode.<br>
